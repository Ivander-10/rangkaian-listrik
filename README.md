Kode HTML ini merupakan program sederhana untuk mensimulasikan rangkaian listrik dengan beberapa lampu yang dapat dinyalakan dan dimatikan menggunakan tombol saklar. Struktur halaman dimulai dengan deklarasi <!DOCTYPE html> yang menandakan bahwa dokumen menggunakan HTML5. Pada bagian <head> terdapat pengaturan karakter UTF-8, judul halaman “Simulasi Rangkaian Listrik”, serta bagian <style> yang berfungsi untuk mengatur tampilan halaman. CSS tersebut mengatur tampilan body agar menggunakan font sederhana, latar belakang berwarna abu-abu muda, dan teks berada di tengah. Elemen .container digunakan sebagai kotak pembungkus rangkaian yang memiliki latar belakang putih, sudut melengkung, serta bayangan agar terlihat seperti kartu. Selain itu, ukuran gambar lampu diatur menjadi 100px dan tombol diberi padding, margin, serta warna khusus untuk tombol Semua ON (kuning) dan Semua OFF (merah).
Pada bagian <body> terdapat dua buah rangkaian lampu. Rangkaian pertama berisi 3 lampu dengan id l1, l2, dan l3, sedangkan rangkaian kedua berisi 4 lampu dengan id la, lb, lc, dan ld. Setiap lampu ditampilkan menggunakan elemen <img> yang merepresentasikan gambar lampu. Di bawah gambar terdapat beberapa tombol yang berfungsi sebagai saklar untuk mengontrol lampu. Tombol “Saklar 1”, “Saklar 2”, dan seterusnya akan memanggil fungsi JavaScript toggleLampu() untuk menyalakan atau mematikan satu lampu tertentu. Selain itu terdapat tombol Semua ON dan Semua OFF yang memanggil fungsi semuaLampu() untuk mengubah kondisi beberapa lampu sekaligus. Pada rangkaian kedua juga terdapat tombol tambahan seperti Grup AB ON dan Grup CD ON yang berfungsi menyalakan lampu dalam kelompok tertentu.
Pada bagian <script> terdapat dua fungsi JavaScript yang mengatur logika program. Fungsi toggleLampu(id) digunakan untuk mengubah kondisi lampu secara bergantian (toggle). Fungsi ini mengambil elemen lampu berdasarkan id, kemudian memeriksa apakah gambar lampu sedang dalam keadaan mati (bulboff). Jika mati maka gambar diganti dengan lampu menyala (pic_bulbon.gif), dan jika menyala maka diganti menjadi lampu mati (pic_bulboff.gif). Sementara itu fungsi semuaLampu(aksi, daftarId) digunakan untuk mengontrol beberapa lampu sekaligus. Fungsi ini menerima parameter aksi (on atau off) serta daftar id lampu dalam bentuk array. Program kemudian menggunakan forEach untuk mengubah gambar setiap lampu dalam daftar tersebut menjadi lampu menyala atau mati sesuai dengan aksi yang dipilih.
